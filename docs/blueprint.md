## Core Features:

- AI-Powered Station Curator: Recommends radio stations to users based on their listening habits and preferences, identified from their interaction history tool
- Automated DJ Messages: Uses AI to generate short, context-aware messages and station identifications to play between songs, dynamically adjusting to the station's genre and mood.
- Live Audio Streaming: Plays welcome messages. Select and stream music from a variety of experimental radio stations in real-time.
- Interactive Audio Visualizer: Presents sound frequencies via audio visualizations.
- Station Information Display: Shows station details, such as name, location, and genre.

## Style Guidelines:

- Primary color: Deep violet (#552B81) to convey sophistication and creativity.
- Background color: Light gray (#F5F3F6), providing a neutral backdrop.
- Accent color: Soft violet (#9A6CC3) is used as a secondary color for interactive elements.
- Body font: 'Inter' (sans-serif) for a clean, modern reading experience.
- Headline font: 'Space Grotesk' (sans-serif) for headlines; conveys a tech-driven vibe.
- Use minimalist icons that clearly represent each function and radio station genre.
- Implement smooth transitions when switching between stations or toggling audio visualizations.
